Etiquetas semánticas

Las etiquetas semánticas sirven para definir la estructura de un sitio web, estas mejoran la creación de una página web y ayudan a que nuestra página web, sea indexada con una exactitud mayor por los motores de búsqueda. (Lluch & Oliva, 2021). z

¿Qué son?

Semántica: Según la definición de la real academia de la legua española, hace referencia a la pertenencia o relativo a las palabras, esto es, el HTML5, las que dan significado a las partes del documento, indican que es el contenido que contienen. Existe un conjunto de etiquetas semánticas fundamental para la maquetación de páginas, estas son las que nos ayudan a definir el esquema principal de una página web, HEADER, NAV, SECTION, FOODER, ASIDE, entre otras que nos indican que es el contenido de nuestra página web y su relación con el conjunto de elementos de HTML.

Para que un motor de búsqueda sepa que partes de del contenido de un sitio web corresponden a cada una de las partes, es necesario hacer uso de las etiquetas semánticas estructuradas, por lo general un sitio web está estructurado por una cabecera, un cuerpo y un pie de página.

En HTML5, podemos encontrar las siguientes etiquetas semánticas estructuradas:

HEADER:Engloba los elementos correspondientes a la cabecera de la página.

FOOTER:Engloba todos los elementos correspondientes al pie de página.

SECTION, ARTICLE, ASIDE:Engloban los elementos que corresponden al cuerpo de la página.

NAV:Esta etiqueta representa los elementos correspondientes a una barra de navegación.
