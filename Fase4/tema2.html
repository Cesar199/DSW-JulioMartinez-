Listas y enlaces

Las listas en HTML5, nos permite crear un conjunto de elementos en listados dentro de una página. Los tipos de listas en HTML5 son los siguientes:

Listas ordenadas

Estas listas nos muestran los elementos de la lista en orden, para dicho orden, tendremos cada elemento de la lista enumerado, ya sea por un numero o una letra.

Para estas listas en HTML, utilizaremos el elemento OL y para cada uno de los elementos, utilizaremos el elemento li.

Listas desordenadas

Este tipo de listas nos permite mostrar los elementos que las conforman sin ningún tipo de orden. Para este tipo de listas se utiliza el elemento ul, posterior se utiliza li como en el anterior caso para definir cada uno de los elementos.

Listas de definiciones

Con estas listas, podemos montar una estructura en HTML, en la que tenemos un valor y una definición. Se definen mediante el elemento dl para la definición y dt para el valor.

Listas anidadas

Se utilizan para anidar unas listas dentro de otras listas seguido del elemento li dentro de las listas.

Enlaces

Los enlaces en HTML5, permiten la navegación por medio de hipervínculos, estos están marcados por la etiqueta a /a con su atributo principal href=”” en donde se escribe la ubicación del archivo de destino.

Tipos

Internos: Enlaces principales de un sitio. Se pueden utilizar como anclas hacia un punto determinado dentro de la misma página.

Externos: Enlaces entre diferentes sitios web. Para especificar el enlace al que queremos dirigir, debemos especificar la ruta absoluta del sitio web, es decir la url completa.
